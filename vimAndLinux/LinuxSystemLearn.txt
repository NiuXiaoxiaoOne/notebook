c + a + f1-f6   字符界面／图形界面切换

c + a + T 终端快捷键
右键 ＋ e 也可以

c + d 重新加载终端／关闭终端
a’ + f4 也可关闭终端
c’ + c 强行终止，重新开始

c’ + 空格 中英输入法切换

c’ + l/输入clear  清屏

tab  自动补全

两下tab  列出可能的命令

上下键，调用已经用过的命令；history调出使用命令的历史记录

c’ + 左右键 跳单词

esc + “.” 调用上一个命令后面的参数

选中后，控制滚轴键 复制并粘贴

c’ + s’ + t  多个终端
c’ + s’ + up/down/空格 切换终端
exit 关闭终端
home/end  第一页／最后一页

date 返回当前时间

whatis + 命令 查询命令的功能
命令 ＋ —-help 返回详细的命令功能
eg: ls [选项] … [文件]

man + 命令：
	“/” + 对象  查找功能
	q 退出
简单的说man比help更加美观
man -k 关键词 返回含有关键词的命令以及功能介绍
info + 命令 也具有查询命令的功能

nano + 文件名.后缀 创建新文件／读取
who 有谁在线

shutdown指令：

+ -h now 立即关机，时间为0
+ -h 20:25  在指定时间关机
+ -h + 10   10分钟后关机
+ -r now 立即重启
+ -c 取消指令

reboot +  回车  立即重启

poweroff + 回车 立即关机
